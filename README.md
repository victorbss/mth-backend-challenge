# REST API endpoint - Coding Challenge

## Depedencies
- PHP 7+ 
- composer

## Installation and Built-in API
clone the repo and run the commands in project root dir:
- composer install
- php -S localhost:8000 -t public 
    - Endpoint /products

## Tests
run the command in project root dir:
- vendor/bin/phpunit
